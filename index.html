Python Snake Game

How to Play:

  1. Download the Python file using the button below
  2. Make sure you have Python and Pygame installed on your computer
  3. Run the game using Python
  4. Use arrow keys to control the snake
  5. Collect food to grow longer
  6. Avoid hitting walls and yourself

Requirements:

  • Python 3.x
  • Pygame library (install using: pip install pygame)
